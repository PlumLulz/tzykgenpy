# tzykgenpy
Turkey Zyxel Keygen

Default ESSID is TTNET_ZyXEL_XXXX or TurkTelecom_XXXXX

Zyxel VMG3312-B10B and VGM3313-B10A

Usage: python3 tzykgen.py S150Y13068675
